# self-introduction-practice

## 概要

COACHTECH 教材 Tutorial 5-1-7 「HTMLの基礎-ハンズオン練習」で作成した成果物「自己紹介」です。

## 使用技術

- HTML5

## 学んだこと

・HTMLの基本構造たる DOCTYPE、html、head（表示されない）、body（表示される）の役割
・body内に入力する見出し（h1~h2）の使い分けと、段落やリストの使い方
・フォーム内の基本構造とlabelやrepuiredの使い方と必要性

## 動作確認

１　VSCodeのエクスプローラーでself-introduction.htmlを右クリックし、統合ブラウザで開くを選択
２　URLをクリックし、新しいタブを開くことを確認した。
